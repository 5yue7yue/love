# 文武

今天在泉州最后一天，去了关帝庙，也去了文庙，在文庙里，大殿中，孔子前，板凳上，一边静心，一边和她聊天，聊了事业，聊了情感。一个多小时，很美好，庄重的环境里，有着她的温馨，这种感觉真好，我说我们是同窗，是同学，她说我说是就是。我说在人生的大学里，我们肯定是同学。
下午我到机场，她问我这么早去机场干嘛，我说想早点把跟她的距离从几千公里变成几百公里。对了还有我在古玩街里拍了一个花瓶的图片发给她看，我觉得她会喜欢，因为我也喜欢，我本来想一百买下，可惜卖家不卖，非要三百，我网上买了一个，到货了再和她说。其实我想送给她，不知道她会不会收。
