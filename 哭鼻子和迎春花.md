# 哭鼻子和迎春花

早晨她说她看了一个视频，哭的稀里哗啦的，一个骑行up的叫悟空的猫去了猫星。她是一个容易被感动的人，也是有强烈同理心的人。

我对她说"有你真好"，她问我为什么会这么说。我回答说"你很像我，很多情感也相通，你的所有感觉就像我也经历一样，让我的感情有了一个安定的归宿，跟我念叨这，念叨那的，让我很安心。"。

我把昨天的日记截图给她看。她说会再见面的，新春花迎春花该开了。

后来她问我如果她像悟空一样离开，我会在某个时刻想起她么。我说会的，我会每天都想起她。我看到蓝蓝的天会想起她，听到雨声会想起她，看到柳枝变绿会想起她，看到实事求是会想起她，春风划过脸颊会想起她，秋天的云儿飘过会想起她。

傍晚她拍了一张迎春花开的照片，说迎春花开了。

我拍了一个太阳发给她。

是该见一面了~
